# Lemon Squeezy (lemon-squeezy)

Lemon Squeezy is on a mission to make selling digital products easy-peasy. As a merchant of record, Lemon Squeezy handles the global tax, compliance, fraud, and payment infrastructure so creators selling digital goods can focus on building.

## APIs

### Lemon Squeezy API

Welcome to Lemon Squeezy! Whether you're launching a brand new store, a newsletter, or you're interested in using Lemon Squeezy as your ecommerce platform, you can find everything you need to know here. The core API exposes stores, products, variants, prices, files, checkouts, customers, orders, subscriptions, discounts, license keys, usage records, and webhooks.

- **Documentation:** https://docs.lemonsqueezy.com/api
- **OpenAPI:** [openapi/lemon-squeezy-api-openapi.yml](openapi/lemon-squeezy-api-openapi.yml)

#### Tags

- Checkouts, Customers, Licenses, Orders, Products, Stores, Subscriptions, Webhooks

### Lemon Squeezy License API

The License API allows you to manage licenses generated by Lemon Squeezy, including activating, deactivating, and validating license keys, and managing instances tied to a key.

- **Documentation:** https://docs.lemonsqueezy.com/api/license-api
- **OpenAPI:** [openapi/lemon-squeezy-license-api-openapi.yml](openapi/lemon-squeezy-license-api-openapi.yml)

#### Tags

- Instances, Keys, Licenses
